== Description ==

= Only for development! =

Displays:

* Header
* Thumbnails
* The Loop (content)
* Footer
* Included template files

Only administrators can use this plugin.

Notice: uses regular expressions and `eval()` to get the current template apart.

TODO: sidebar, search-form, other template parts.

== Installation ==

This section describes how to install the plugin and get it working.

1. Uppack the ZIP to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
